jst do this.

Just type what you want done. jst turns it into a shell command and runs it.

$ jst find all files bigger than 500 mb in ~/downloads
→ find ~/downloads -type f -size +500M

Free for reasonable use* · or BYOK · open source

↓

01 / why this exists

My shell skills
got rusty.

With coding agents doing most of the work, I’ve found myself using the shell less. Then, when I do need it, I no longer remember how to do basic things.

Sure, I can ask an LLM how to find a file, then copy its answer into the terminal. But I kept thinking: why not just ask the terminal?

So I made jst. Say what you need after the command; it turns that into shell, shows you what it will run, and runs it. It asks before obvious destructive stuff. --yolo is there for more confident days.

$ jst find files I changed today

02 / getting started

Off you go.

  1. 01

    Install it.

    brew install yoavf/tap/jst or download the latest release →
  2. 02

    Put it on your PATH.

    Homebrew does this for you. With a downloaded binary, put it somewhere your shell can find it.

  3. 03

    Ask for what you need.

    jst find files I changed today

JST runs the command, or pauses for approval when something looks destructive. Feeling brave? --yolo it.

03 / small print, but nice

Free, within
reason.*

Hosted

Small model,
small bill.

JST queries run on a small, fast model, so they’re cheap to run. The budget should be enough for around 200,000 queries each month. A few rate limits prevent abuse—not normal use.

please don’t ruin it for everyone ♡

or

BYOK

Bring your own key.

Run the open-source server with an OpenAI-compatible provider, then point JST at it. Your key, your limits.

JST_API_URL=http://localhost:8080/translate
